Download the Syllabus


How this website works:

You will find a link for each Class in the menu. There you will find a Lesson tab with the topics covered during class (the powerpoint slides).

Under each Class, you will also find an Exercises tab. Each exercise will link to a Jupyter Notebook; instructions for the exercise are listed inside the notebook.

What to expect out of this class:

While working through this ANALYSIS GUIDE, you will learn about the data analytics workflow and get hands-on exposure to many parts of it, including collecting, cleaning, merging data, exploratory data analysis, communicating, visualizing, and sharing your findings.
  • Python
  • SQL
  • Reproducible Workflow with Jupyter Notebooks
  • Analytics Careers
  • Exploratory Data Analysis
  • Data Visualization & Storytelling

Furthermore:
  • You will be able to ask questions of instruction staff when you need clarification.
  • You will be pointed at resources to help you after the class is over (no matter what your goal is).
